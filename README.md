# DiscordBot
A Moderation/leveling Discord bot, Made for TeaClient Called TeaClientCore.
<br>
<img src="https://raw.githubusercontent.com/TeaclientMinecraft/.github/main/profile/dividers.png">
## Guides

Install the bot:

```bash
bun install
```


Running the bot:

```bash
bun run src/bot.js
```
